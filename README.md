## 💻 Practice Tasks

### 1️⃣ Even or Odd Number

```python
num = 10

if num % 2 == 0:
    print("Even number")
else:
    print("Odd number")
```

---

### 2️⃣ Print Numbers Using Loop

```python
for i in range(1,11):
    print(i)
```

---

### 3️⃣ Calculate Total Marks

```python
marks = [70,85,90,60,75]

total = 0
for m in marks:
    total = total + m

print("Total Marks:", total)
```

---

### 4️⃣ Find Maximum Number

```python
numbers = [4,7,2,9,5]

print(max(numbers))
```
